💰 ERP Payment Module (Vendor Bill Settlement)

This project is a React-based Payment Management Module inspired by ERP systems. It allows users to manage vendor bills, perform full or partial payments, and track outstanding balances efficiently.

🚀 Features

  ✅ Vendor selection and filtering
  ✅ View outstanding bills
  ✅ Full & Partial payment options
  ✅ Real-time validation and calculations
  ✅ Automatic settlement amount calculation
  ✅ Error handling for invalid inputs
  ✅ Local storage for saving payment records
  ✅ Clean and responsive UI

🧮 Functional Highlights

  Prevents overpayment beyond outstanding amount
  Ensures valid numeric input for partial payments
  Dynamically updates:
    Total Bills
    Outstanding Amount
    Settlement Amount

🛠️ Tech Stack

   React.js
   JavaScript (ES6)
   Bootstrap / CSS
   LocalStorage (for data persistence)


🎯 Use Case
   This module simulates the Accounts Payable / Vendor Payment functionality found in ERP systems, helping manage and settle bills efficiently.
